# Privacy Policy

Memoリー respects your privacy.

## Data Collection
The app collects limited, anonymous data solely for app functionality, including:
- Anonymous user identifiers
- Subscription status
- Usage counters (such as AI usage)

No personal information such as name, email address, or location is collected.

## Data Usage
Collected data is used only to:
- Enable core app functionality
- Manage subscriptions
- Prevent abuse of app features

Data is not used for advertising, tracking, or marketing purposes.

## Data Sharing
Memoリー does not sell or share user data with third parties for advertising or tracking.

## Data Storage
Some anonymous data is stored securely using third-party services (such as Firebase) to support app functionality.
